# 🌦️ Weather Report Application

A responsive weather application built using HTML, CSS, and JavaScript that displays real-time weather information.

## 🚀 Features

- Search weather by city
- Display temperature details
- Weather condition icons
- Sunrise & sunset timing
- Humidity and wind speed
- Responsive UI design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
