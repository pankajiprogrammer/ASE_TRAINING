# 🚀 ASE Training Projects & Learning Resources

This repository contains frontend web development practice projects and learning resources created during ASE Training using HTML, CSS, JavaScript, and Bootstrap.

---

# 📚 Learning Resources

These folders contain practice examples and concept-based learning exercises.

| Sr No | Topic | Description |
|------|------|------|
| 01 | Box Layout | Understanding box model, margin, border, padding, and content |
| 02 | Columns | Multi-column layouts and responsive structure |
| 03 | Display Property | Learning block, inline, inline-block, and none properties |
| 04 | Flex Box Layout | Modern flexible layouts using Flexbox |
| 05 | Float and Clear | Traditional layouts using float and clear |
| 06 | Grid Layout | Advanced responsive layouts using CSS Grid |
| 07 | Position Practice | Static, relative, absolute, fixed, and sticky positioning |
| 08 | Bootstrap | Responsive design using Bootstrap framework |
| 09 | JS Learning | JavaScript basics, DOM, events, functions, arrays, async concepts |

# 📌 Main Projects

## 1️⃣ Weather App

A responsive Weather Application that provides real-time weather information using a weather API.

### 🌦️ Live Demo

🔗 https://pankajiprogrammer.github.io/ASE_TRAINING/TASK/11-weather_App_Task/

### ✨ Features

- Search weather by city name
- Temperature details
- Humidity & Wind Speed
- Sunrise & Sunset timing
- Dynamic weather icons
- Responsive design

### 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API
