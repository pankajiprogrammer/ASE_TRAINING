# 📋 Dynamic Task Manager Application

A modern and responsive **Dynamic Task Manager Application** built using **HTML**, **CSS**, and **JavaScript** that helps users manage daily tasks efficiently with dynamic functionality and interactive UI components.

---

# 📌 Project Overview

The Dynamic Task Manager is designed to help users:
- Add tasks
- Manage task lists
- Remove completed tasks
- Organize daily activities

This project focuses on practicing:
- DOM Manipulation
- Event Handling
- Dynamic Rendering
- JavaScript Logic Building
- Responsive Web Design

---

# 🚀 Features

## ➕ Add New Tasks
Users can add new tasks dynamically using the input field.

---

## 🗑️ Delete Tasks
Tasks can be removed instantly with delete functionality.

---

## ✅ Mark Tasks as Completed
Users can mark tasks as completed for better task tracking.

---

## 🔄 Dynamic UI Updates

Clean and attractive UI with organized task sections.

# ⚙️ Functionalities Explained

## 📌 Task Creation

Users can create tasks dynamically using JavaScript.

Example:

```javascript
addButton.addEventListener("click", () => {
  createTask();
});
```

---

## 🧠 DOM Manipulation

JavaScript dynamically:
- Creates task elements
- Updates UI
- Removes tasks
- Handles task completion

without reloading the page.

---

## 🎯 Event Handling

The project uses event listeners to:
- Add tasks
- Delete tasks
- Handle user interactions

Example:

```javascript
button.addEventListener("click", function () {
  console.log("Task Added");
});
```

---

## 🎨 Responsive Design

CSS media queries are used to ensure proper responsiveness across different screen sizes.

Example:

```css
@media (max-width: 768px) {
  .container {
    width: 90%;
  }
}
```
